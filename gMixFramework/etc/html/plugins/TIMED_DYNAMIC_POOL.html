Timed Dynamic Pool

Output strategy used in Mixminion according to Mixminion spec v1.31 2007/04/21 (last checked on 31.08.2012). Note that the actual Mixminion implementation (31.08.2012) is slightly different. A Java port of the actual (29.08.2012) Mixminion implementation can be found below. Also note that the plug-ins "Timed Cottrell Pool", "Timed Dynamic Pool" and "Mixminion Pool" are basically the same thing, implemented slightly different.

Plugin key:

TIMED_DYNAMIC_POOL

Properties:

  • Send rate (ms):
    • Key: TIMED_DYNAMIC_POOL_SEND_INTERVAL_IN_MS
    • Type: Integer
    • Min: 0

  • Minimum messages in pool (requests):
    • Key: TIMED_DYNAMIC_POOL_MIN_MESSAGES_IN_POOL
    • Type: Integer
    • Min: 0

  • Fraction:
    • Key: TIMED_DYNAMIC_POOL_FRACTION
    • Type: Double
    • Min: 0
    • Max: 1
