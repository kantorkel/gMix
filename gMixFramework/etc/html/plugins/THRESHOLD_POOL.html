Threshold Pool

Waits until x + y messages are collected and than forwards y messages (chosen at random). When "y = 1", same as the "Basic Cottrell Pool"

Plugin key:

THRESHOLD_POOL

Properties:

  • Pool size (requests):
    • Key: THRESHOLD_POOL_MIN_POOL_SIZE
    • Type: Integer
    • Min: 1

  • Threshold:
    • Key: THRESHOLD_POOL_THRESHOLD
    • Type: Integer
    • Min: 1
