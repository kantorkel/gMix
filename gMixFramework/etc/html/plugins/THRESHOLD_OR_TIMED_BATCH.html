Threshold Or Timed Batch

Forwards all messages whenever x messages are batched or every "output interval" ms.

Plugin key:

THRESHOLD_OR_TIMED_BATCH

Properties:

  • Sending Rate (ms):
    • Key: THRESHOLD_OR_TIMED_BATCH_SENDING_RATE_IN_MS
    • Type: Integer
    • Min: 0

  • "Batch Size (requests):
    • Key: THRESHOLD_OR_TIMED_BATCH_BATCH_SIZE
    • Type: Integer
    • Min: 1
